### 遮罩层

* 遮罩层

以下这种方案比较稳妥。可以在遮罩层监听相应的事件。

```css
.overlay {
  position: fixed;
  top: 0;
  left: 0;
  right: 0;
  bottom: 0;
  background: rgba(0,0,0,.8);
}

.show {
  position: absolute;
  z-index: 1;
}
```

### 通过 point-events: none 来禁用触发事件

场景: ImagePicker 组件点击图片禁止其在 app 中弹出底部图片;

`point-events: none` 类似 button 的 disabled 属性, 但是其能涵盖到其它任意属性。
