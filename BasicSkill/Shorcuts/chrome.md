### 快捷键

```
无痕模式: cmd + shift + n
```

### debugger

* `step`: 多步执行;
* `step over`: 单步执行, 遇到子函数会将子函数当成是一个整体执行;
* `step into`: 单步执行, 遇到子函数进入子函数;

### Take Full Size Screenshots

1. Press `Cmd + Shift + P`;
2. Type `Capture full size screenshot`;

### 切换

chrome://flags/#enable-force-dark, enabled 启动暗黑
