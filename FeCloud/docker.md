### Docker 是什么？

Docker 属于 Linux 容器的一种封装，提供简单易用的容器使用接口。它是目前最流行的 Linux 容器解决方案。

### 使用场景

1. 提供一次性的环境。
2. 提供弹性的云服务。
3. 组建微服务架构。

### Install

```js
brew cask install docker
```
